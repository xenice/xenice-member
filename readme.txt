Comprehensive WordPress membership management plugin with WooCommerce free download support.

== Description ==
Xenice Member is a simple and easy-to-use WordPress membership management plugin.
It allows you to create and manage membership levels, assign users to levels, 
set expiration times, and provide exclusive WooCommerce free downloads to members. 
Perfect for websites with paid or free membership content.

**Features:**
* Manage membership levels (Lifetime, Annual, Monthly, or custom)
* Edit and add members through a dedicated admin editor
* Set membership expiration times
* Assign permissions to specific members
* WooCommerce integration for free downloadable products
* User-friendly admin interface with sortable member lists

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/xenice-member` directory, or install the plugin via the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the "Members" menu in the WordPress admin to manage members and membership levels.

== Frequently Asked Questions ==
= How do I assign a membership level to a user? =
Go to the Member Editor in the admin menu, select the user, and choose the membership level.

= Can I offer free downloads for certain products? =
Yes. If the user has the "WooCommerce Free Download" permission, a free download button will appear on downloadable products.
